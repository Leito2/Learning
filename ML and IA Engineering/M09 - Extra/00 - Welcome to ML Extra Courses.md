# 👋 Welcome to ML Extra Courses

## What will you learn?

This supplementary track is designed to bridge the gap between academic machine learning knowledge and the practical skills demanded by industry roles. You will learn how to compete effectively in data science competitions, build production-grade ML pipelines from scratch, and adapt large language models for specialized tasks. Each course combines deep theoretical foundations with hands-on coding exercises, ensuring you can both understand the "why" and implement the "how."

By the end of this track, you will be able to:

- Architect and execute end-to-end machine learning projects using modern MLOps practices
- Compete at a high level on Kaggle by understanding evaluation metrics, cross-validation strategies, and ensemble techniques
- Fine-tune large language models efficiently using parameter-efficient methods and alignment techniques
- Build reproducible, scalable, and monitorable ML systems ready for real-world deployment

## Capstone Project

The capstone for this extra track asks you to select a real-world problem—ideally from your domain of interest—and execute a complete solution using at least two of the three core skills taught here. For example, you might participate in a Kaggle competition using an end-to-end MLOps pipeline with experiment tracking, or fine-tune an open-source LLM for a specific domain and deploy it behind an API with drift monitoring. The goal is to produce a portfolio piece that demonstrates not only modeling skill, but engineering maturity.

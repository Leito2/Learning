# 🌐 Bienvenida - Open Source y Comunidad

## ¿Qué aprenderás en este curso?
- Cómo contribuir efectivamente a proyectos de código abierto en el ecosistema de [[Machine Learning]] e [[Inteligencia Artificial]]
- Estrategias para construir y mantener comunidades técnicas vibrantes y sostenibles
- El proceso completo de publicación de librerías Python y papers de investigación
- Planificación de carrera profesional en ML/IA: desde ingeniero hasta líder técnico
- Un caso práctico integral para lanzar tu propia librería open source desde cero

## Proyecto final
El proyecto final consiste en diseñar, documentar y publicar una librería open source mínima pero funcional orientada a ML/IA. Deberás crear un repositorio con código limpio, tests automatizados, documentación en ReadTheDocs o MkDocs, y una estrategia de comunidad inicial. Este proyecto integra todos los módulos del curso: contribución, comunidad, publicación y visión de carrera.

---

💡 **Consejo:** Open source no es solo código; es relaciones humanas. Construye comunidad antes de construir software.

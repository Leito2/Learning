## 🤔 ¿Por qué PostgreSQL?

PostgreSQL es un sistema de gestión de bases de datos relacional (RDBMS) de código abierto. Es conocido por:

- **Cumplimiento con SQL estándar** (muy estricto).
- **Extensibilidad**: soporta tipos de datos personalizados, funciones, extensiones.
- **Rendimiento y escalabilidad** para proyectos pequeños y empresariales.
- **Soporte nativo para JSON, arrays, GIS**, y más.

> 💡 **Dato:** PostgreSQL es la base de datos preferida en muchos proyectos de ciencia de datos e ingeniería de ML.

## 📋 Glosario rápido

| Término | Significado |
|---------|-------------|
| **RDBMS** | Sistema de gestión de bases de datos relacionales |
| **DDL** | Data Definition Language (CREATE, ALTER, DROP) |
| **DML** | Data Manipulation Language (INSERT, UPDATE, DELETE) |
| **DQL** | Data Query Language (SELECT) |
| **Schema** | Esquema: agrupación lógica de tablas y objetos |

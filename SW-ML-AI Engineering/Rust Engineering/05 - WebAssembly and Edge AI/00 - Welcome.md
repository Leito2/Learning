# 🌐 Welcome to WebAssembly and Edge AI

## What will you learn?

- **WebAssembly fundamentals**: Architecture, compilation pipeline, and Rust integration
- **Browser-based ML**: Running inference with TensorFlow.js, candle-wasm, and WebGPU
- **Serverless edge computing**: WASI, Cloudflare Workers, Fermyon Spin
- **Performance analysis**: WASM vs native vs JavaScript benchmarks
- **ML pipeline design**: Building production-ready inference pipelines in WASM
- **Edge deployment**: Real-world case studies from Tesla, Cloudflare, and Spotify

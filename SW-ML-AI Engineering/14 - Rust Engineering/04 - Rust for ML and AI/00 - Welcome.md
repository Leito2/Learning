# 🤖 Welcome to Rust for ML and AI

## What will you learn?

- How to bind Python and Rust using PyO3 for high-performance ML pipelines
- Using Candle, HuggingFace's minimalist ML framework, to run models natively in Rust
- Deploying models via ONNX Runtime with Rust bindings for cross-platform inference
- Comparing ONNX Runtime vs Candle to choose the right runtime for your deployment target
- Mastering advanced ONNX patterns: graph optimization, custom operators, quantization pipelines
- Building high-throughput inference servers with Actix-web, Axum, and tonic
- Integrating vector databases (Qdrant, pgvector) for semantic search and RAG applications
- Applying model quantization and optimization for edge deployment and faster inference

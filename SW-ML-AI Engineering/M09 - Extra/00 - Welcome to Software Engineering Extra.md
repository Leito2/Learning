# 👋 Welcome to Software Engineering Extra

## What will you learn?

This advanced module deepens your expertise in production-grade machine learning systems. You will master the engineering practices that separate experimental notebooks from scalable, reliable AI services deployed to millions of users.

- Build high-throughput ML APIs with modern async Python frameworks
- Design distributed systems that survive real-world traffic and data scale
- Implement robust testing strategies for data pipelines and model serving
- Automate model delivery with GitOps, canary deployments, and feature flags
- Leverage Rust for performance-critical ML infrastructure components

Each course blends systems thinking with hands-on code, mirroring the rigor of the existing [[../Docker Profesional|Docker Profesional]] track.

## Capstone Project

By the end of this module, you will architect and deploy an end-to-end real-time recommendation API. The system will ingest user events, serve personalized rankings through a FastAPI gateway, validate models with automated tests, deploy via GitOps pipelines, and offload heavy embedding retrieval to a Rust microservice. This project demonstrates senior ML engineering competency across the entire stack.

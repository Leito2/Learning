# 🚀 Welcome to Advanced Rust

## What will you learn?

This course pushes you beyond safe, synchronous Rust into the territories that define senior systems engineers. You will master asynchronous concurrency with [[01 - Concurrency with Tokio and Async-Await|Tokio and async/await]], learning how Rust's type system enforces thread safety at compile time. You will explore [[02 - Unsafe Rust and FFI|unsafe blocks and FFI boundaries]], understanding when and how to bypass the borrow checker responsibly. You will write [[03 - Macros and Metaprogramming|declarative and procedural macros]], generating code at compile time to eliminate boilerplate. You will dive deep into [[04 - Memory Management Deep Dive|smart pointers and interior mutability]], managing complex ownership graphs without leaks. You will build robust pipelines using [[05 - Testing, Benchmarking, and Profiling|tests, benchmarks, and profilers]], ensuring your code is both correct and fast. Finally, you will ship production artifacts through [[06 - Cross-compilation and Deployment|cross-compilation and containerized deployment]], targeting everything from Linux servers to WASM edge workers.

## Capstone Project

By the end of this course, you will build **TurboCache** — a high-performance, async-capable in-memory key-value store with pluggable backends, comprehensive test suites, cross-platform deployment, and a declarative configuration macro. This project combines every module into a single, deployable systems component.
